# sukanya-3d-portfolio

Production-ready single page 3D portfolio for Sukanya Beuria, built with React 19, Vite, Tailwind CSS v4, Framer Motion, GSAP, Three.js, React Three Fiber, Drei, Lenis, React Icons, Lottie React, EmailJS, and Recharts.

## Features

- Luxury black, purple, pink, and violet visual system
- Aurora background, floating particles, stars, cursor glow, 3D scene, and scroll progress
- Framer Motion and GSAP scroll reveal animations
- Lenis smooth scrolling
- Magnetic buttons, hover tilt, glass panels, and animated borders
- Hero, About, Skills, Projects, Experience Timeline, GitHub, Achievements, Certifications, Tech Orbit, Contact, and Footer sections
- EmailJS-ready contact form with safe placeholder behavior
- Responsive, accessible, SEO-friendly structure with code-split sections

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## EmailJS Setup

Create a `.env` file and add:

```bash
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

## Assets

Public placeholders are stored in `public/assets`. Replace `resume.pdf` with the final resume and project SVGs with final screenshots when available.
